# OOAD

面向对象分析与设计 (Object-Oriented Analysis and Design) 课程项目

## 项目简介

本项目包含 OOAD 课程相关的作业、实验和设计文档，主要涉及面向对象编程、软件设计模式等内容。

## 目录结构

```
.
├── client/              # 客户端代码
├── server/              # 服务器端代码
├── information/         # 信息采集模块
├── ooad-assign2_2/      # 作业 2-2
├── LICENSE              # 许可证文件
└── .gitignore
```

## 项目内容

- **client/** - 客户端应用程序
- **server/** - 服务器端应用程序
- **information/** - 信息采集相关模块
- **ooad-assign2_2/** - OOAD 作业 2-2

## 技术栈

- Python
- 面向对象设计模式
- 网络编程

## 使用方法

1. 克隆本仓库
2. 根据具体子项目的说明文档运行相应代码
